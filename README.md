# EC Language
## Make Programming EC

## Instructions

First build this project. Here we use CMake:

```bash
$ mkdir build && cd build  # Create a CMake workspace
$ cmake .. && make
$ ./eclang # Start eclang
```

Running Examples

```
$ ./eclang ../examples/hello-world.kids
Hello, world!
```
